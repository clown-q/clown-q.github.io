# 缓冲流

缓冲是用来在两种不同速度的设备之间传输信息时平滑传输过程的常用手段，主要是为了解决 cpu 和 IO 设备之间速度不匹配的问题，提高 cpu 和 IO 设备的并行性，这句话相比上课的时候会经常性的听到，Java 对其也有相对应的支持

使用的方法，没有太大的变化，实例化出对象的时候会用 BufferedInputStream 进行一个包装
