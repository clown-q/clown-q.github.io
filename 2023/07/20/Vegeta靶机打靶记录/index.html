# 前言

前两天坐牢被折磨的有点惨，找一台简单一点的靶机来霍霍一下乐呵乐呵，但是不幸的是找到了一台让我有些痛苦的靶机，做完后我感觉这可能是一个 misc 题

# 靶机描述

靶机难度：低级

目标：root 权限的 shell

涉及的攻击方法：

  • 主机发现
  • 端口扫描
  • 信息收集
  • 路径枚举
  • 摩尔斯码
  • 数据编码还原
  • 二维码解码
  • 本地提权

虚拟机：

  • 格式： 虚拟机 （Virtualbox OVA）
  • 操作系统： Linux.

联网：

  • DHCP 服务：已启用
  • IP 地址自动分配

这在 VirtualBox 而不是 VMware 上效果更好。
