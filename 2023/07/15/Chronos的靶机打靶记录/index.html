# 前言

这不刚写完 nodejs 的相关利用，这里也是马上找到了一个 js 的靶机来学习一下，本次的的目标 Chronos: 1 ~ VulnHub，本来想水一下描述，但是发现作者就给了一个中等难度，那我就自己做下来的过程给一个描述。

# 描述

难度：中级

设计的任务：

  • 主机发现，端口扫描
  • WEB 应用攻击
  • 命令注入
  • 数据编解码
  • 框架漏洞利用
  • 代码审计
  • 本地提权

虚拟机：

  • 格式： 虚拟机 （Virtualbox OVA）
  • 操作系统： Linux.

联网：

  • DHCP 服务：已启用
  • IP 地址自动分配
