# 前言

知道 docker 这个小东西的优点后，我一直在想，如果我将环境部署到 docker 中是不是就很安全，但是哪有绝对的安全，docker 逃逸？

# 靶机描述

难度：高

目标：拿到 root 权限

涉及的攻击方法：

  • 主机发现
  • 端口扫描
  • WEB 信息收集
  • FTP 服务攻击
  • 缓冲区溢出
  • 模糊测试
  • 漏洞利用代码编写
  • 流量转包分析
  • 堆溢出漏洞攻击
  • Metasploit（MSF）
  • 手动修复 EXP 代码
  • 本地提权

虚拟机：

  • 格式： 虚拟机 （Virtualbox OVA）
  • 操作系统： Linux.

联网：

  • DHCP 服务：已启用
  • IP 地址自动分配
