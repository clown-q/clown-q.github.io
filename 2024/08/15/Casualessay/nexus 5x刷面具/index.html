# 准备

一台 everything 的 kali

twrp 的下载地址：https://eu.dl.twrp.me/bullhead/twrp-3.2.3-0-bullhead.img.html

Magisk 的下载地址：https://github.com/topjohnwu/Magisk/releases

# 开始折腾

首先使用数据线将这个手机连接到电脑上

然后再这个虚拟机这里选着可移动设备将设备连接到虚拟机上，手机开启 usb 调试模式

adb devices -l 是用于显示连接到您的电脑上的 Android 设备的命令。该命令会列出设备的详细信息，包括设备 ID、型号、产品和设备的状态。

上传 Magisk 至手机目录

然后，先进到进入 fastboot 模式（机器人界面）

刷入 Twrp Recovery

使用 +- 音量键进入 Recovery mode

然后按开机键

# 刷入 Magisk

文件在前面已经放进去了，这里直接去安装即可

这里点击这个 install

然后选择这个文件

成不成就看这一划拉

齐活了
